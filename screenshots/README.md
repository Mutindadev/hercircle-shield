# Screenshots Folder

This folder contains screenshots of the HerCircle Shield Chrome Extension for documentation purposes.

## Recommended Screenshots to Include:

1. **extension-icon.png** - Purple heart extension icon in Chrome toolbar
2. **home-tab.png** - Main dashboard with panic button and stats
3. **circles-tab.png** - Trusted contacts and support circles
4. **learn-tab.png** - Educational resources and safety guides
5. **settings-tab.png** - Extension settings and preferences
6. **panic-modal.png** - Panic button confirmation modal
7. **evidence-capture.png** - Evidence capture feature
8. **threat-detection.png** - Real-time threat detection on social media
9. **notifications.png** - Extension notifications
10. **educational-modal.png** - Educational content modal

## File Naming Convention:

Use descriptive, lowercase names with hyphens:
- ✅ `panic-button-modal.png`
- ✅ `threat-detection-twitter.png`
- ❌ `Screenshot 2025-11-29.png`

## Image Format:

- **Format:** PNG (preferred) or JPG
- **Size:** Keep under 500KB per image
- **Resolution:** 1920x1080 or smaller

Once you add screenshots here, they will be automatically referenced in the README.md!
